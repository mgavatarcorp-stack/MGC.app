# Project Zooter

## Overview
Project Zooter is a coordination protocol to direct a decentralized memetic army with a meme-backed currency governed by pseudonymous memelord shitposters.

## Features
- Decentralized governance
- Proof of Humanity verification
- Token-based incentive system
- Dashboard with top contributors and communities
- Bounty board and leaderboards

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Deploy smart contracts: `truffle migrate`

## Folder Structure
- `contracts/`: Smart contracts
- `src/`: Frontend source code
- `public/`: Public assets
- `scripts/`: Deployment and interaction scripts
- `test/`: Smart contract tests
